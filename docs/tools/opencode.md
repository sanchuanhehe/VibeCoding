# OpenCode

## 工具定位

- 定位：开源 AI coding agent
- 入口：CLI / Desktop（Beta）

## 功能列表

```{list-table}
:header-rows: 1

* - 能力
  - 状态
  - 说明
* - 开源
  - ✅
  - 代码可审计
* - Provider-agnostic
  - ✅
  - 支持多模型供应商
* - Build/Plan agent
  - ✅
  - 内置基础代理能力
* - 可扩展配置
  - ✅
  - 适合工程化定制
```

## 适合场景

- 偏好开源和终端工作流、希望避免绑定单一模型厂商的开发者/团队

## 主要限制/注意点

- 以终端体验为核心，企业治理与托管能力需结合外部平台；部分图形化体验仍在完善

## 详细评价

### 优势

- 开源 + 终端工作流，灵活性高。

### 局限

- 企业托管能力需外部平台配合。
