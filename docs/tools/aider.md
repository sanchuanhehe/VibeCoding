# Aider

## 工具定位

- 定位：终端 AI Pair Programming（开源）
- 入口：CLI

## 功能列表

```{list-table}
:header-rows: 1

* - 能力
  - 状态
  - 说明
* - Repo map
  - ✅
  - 仓库上下文组织能力
* - Git 自动提交
  - ✅
  - 集成版本管理流程
* - Lint/Test 联动
  - ✅
  - 可接质量检查
* - 多模型支持
  - ✅
  - 云端与本地模型可选
```

## 适合场景

- 终端优先、重视可控改码与 Git 流程

## 主要限制/注意点

- 体验偏 CLI 心智，对纯 IDE 用户不一定友好

## 详细评价

### 优势

- 终端流与 Git 流程结合紧密。

### 局限

- 对纯 IDE 用户不够友好。
