# Amazon Q Developer

## 工具定位

- 定位：生成式 AI 开发助手（AWS 深度集成）
- 入口：IDE 插件 / CLI / AWS 控制台 / 聊天应用

## 功能列表

```{list-table}
:header-rows: 1

* - 能力
  - 状态
  - 说明
* - 代码聊天与补全
  - ✅
  - 日常开发支持
* - 代码生成与扫描
  - ✅
  - 兼顾效率与安全
* - 重构升级
  - ✅
  - 支持迁移/改造场景
* - AWS 运维辅助
  - ✅
  - 云上开发运维联动
```

## 适合场景

- AWS 生态内开发与运维一体化团队

## 主要限制/注意点

- 最佳价值在 AWS 场景，跨云与非 AWS 工作流下能力优势可能下降

## 详细评价

### 优势

- AWS 场景下协同价值高。

### 局限

- 跨云场景优势会下降。
