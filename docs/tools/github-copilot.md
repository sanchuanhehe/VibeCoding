# GitHub Copilot

## 工具定位

- 定位：通用 AI coding + review
- 入口：GitHub / IDE / Terminal

## 功能列表

```{list-table}
:header-rows: 1

* - 能力
  - 状态
  - 说明
* - 代码补全
  - ✅
  - IDE 内联补全
* - Agent / 自动任务
  - ✅
  - Agent mode 与编码代理能力
* - PR 审查
  - ✅
  - 可用于代码评审场景
* - 企业治理
  - ✅
  - 支持组织配置与管理能力
```

## 适合场景

- 已深度使用 GitHub 生态的团队

## 主要限制/注意点

- 价格与配额受套餐影响(额度按次计费,还算便宜,但是有最大限额)；企业能力需管理员配置

## 详细评价

### 优势

- 生态成熟、覆盖面广 vscode 上游支持,交互逻辑清晰

### 局限

- 配额与能力受套餐影响。
