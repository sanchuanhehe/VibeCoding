# CodeBuddy

## 工具定位

- 定位：全流程 AI 编程平台（IDE/插件/CLI）
- 入口：CodeBuddy IDE / IDE 插件 / CLI

  - ✅
  - 研发环节覆盖
* - 规则/记忆/MCP
  - ✅

* - 企业版能力
  - ✅

## 适合场景

- 国内团队、腾讯云生态、需要企业管理能力的组织

## 主要限制/注意点

- 产品线较多（IDE/插件/CLI），落地时需先统一团队工作模式；“Flow”命名在公开文档以模式能力呈现为主

## 详细评价

### 优势

- 国内企业协作与平台化能力较强。

### 局限

- 产品形态较多，落地需统一使用规范。
