# Sourcegraph Cody

## 工具定位

- 定位：企业代码库上下文强化的 AI coding assistant
- 入口：VS Code / JetBrains / Visual Studio / Web / CLI

## 功能列表

```{list-table}
:header-rows: 1

* - 能力
  - 状态
  - 说明
* - 代码库上下文
  - ✅
  - 强调多仓库语义理解
* - Chat / Auto-edit
  - ✅
  - 日常开发辅助
* - Prompts / Context Filters
  - ✅
  - 细粒度上下文控制
* - 企业接入
  - ✅
  - 与 Sourcegraph 生态协同
```

## 详细评价

### 优势
- 大型代码库场景适配好。

### 局限
- 与 Sourcegraph 平台耦合较高。
